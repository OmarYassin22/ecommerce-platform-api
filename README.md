# 🛒 E-Commerce Platform API

A scalable and production-ready e-commerce backend built using ASP.NET Core.  
This project provides a complete solution for managing products, users, orders, and payments.

---

## 🚀 Features

- 🔐 JWT Authentication & Role-Based Authorization
- 🛍️ Product & Category Management
- 🧾 Order & Checkout Workflow
- 💳 Stripe Payment Integration
- ⚡ Redis Caching for performance optimization
- 📦 Clean Architecture (Separation of concerns)
- 📊 Swagger API Documentation

---

## 🛠️ Tech Stack

- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Redis
- Stripe
- Swagger

## ⚙️ Getting Started

### 1. Clone the repository
git clone https://github.com/OmarYassin22/ecommerce-platform-api.git
### 2. Configure appsettings.json
Add your SQL Server connection string
Add Stripe API keys
Configure Redis connection
### 3. Run the project
dotnet run
📬 API Endpoints
Auth (Register / Login)
Products (CRUD)
Orders (Create / Track)
Payments (Stripe Integration)
